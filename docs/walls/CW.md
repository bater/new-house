## 設計決策

**已確認**：
- ✅ **已購 Hefeng KL Board 壁掛翻轉床**（Murphy Bed，詳 [產品頁](../references/products/#kl-board--壁掛翻轉床--書桌murphy-bed)）
  - 床收起時變書桌，翻下變床
  - 需預留 **深度 106 cm** 給書桌翻轉空間
  - 床架離地 26 cm、桌面離地 72 cm

**架高夾層配置（上下層）** — 詳見 [架高工程 NT$56,000](../contract/estimate/#十架高工程)：

- **上層 = 儲藏空間**（在 Murphy 床與書桌正上方）
  - [ ] **不要壓迫** — 夾層高度讓下方床 + 書桌有足夠淨空
  - [ ] **要明亮** — 下方採光、淺色飾面、燈光
  - [ ] **不露雜物** — 儲藏面需有門片 / 布簾 / 推拉門遮擋
- **下層 = 床 + 書桌**（Hefeng KL Board 翻轉 Murphy Bed）

**待決**：
- [ ] 選 KL 90 (197×80) 還是 KL 120 (197×104)
- [ ] 是否加購選配：LED 燈、床頭片、原廠床墊
- [ ] 床身結構固定方式（RC / 輕隔間補強）
- [ ] 書桌椅（Murphy 展開後椅子的收納位置）
- [ ] 上層儲藏取物方式（樓梯 / 踏階 / 爬梯）
