## 天花板：盡量不做

**保留原始樓高最大化**，C 房**盡量不做裝飾天花**：
- [ ] 候選：**直接油漆原始 RC 樓板**（裸頂刷白 / 淺色）
- [ ] 候選：**貼頂照明**（吸頂燈 / 貼頂線燈）— 不壓樓高
- [ ] 架高夾層上層儲藏的高度 → 更有空間
- [ ] [F 一對三冷氣](F/#空調系統一對三室外機在-f) 室內機進 C 的管線走明管 / 牆側收邊

- [ ] **前提**：**不破壞建物外觀**
