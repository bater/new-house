## 設計決策

**C 房的設計重點全集中在此面牆** — 翻轉單人床 + 架高夾層收納，CE 牆則 [留白保持通風採光](CE)。

**已確認**：
- ✅ **已購 Hefeng [KL 120 Board 壁掛翻轉床](../references/products/#kl-board--壁掛翻轉床--書桌murphy-bed)**
  - 床收起後能**降下書桌板**作為工作 / 閱讀區
  - 床板翻下 = 睡眠；床板收起 + 書桌板降下 = 書桌
  - 需預留 **深度 106 cm** 給書桌翻轉淨空
  - 床架離地 26 cm、書桌面離地 72 cm

**架高夾層配置（上下層）** — 詳見 [架高工程 NT$56,000](../contract/estimate/#十架高工程)：

- **上層 = 儲藏空間**（在 Murphy 床與書桌正上方）
  - [ ] **層架高度設計關鍵**：層架下緣**必須高於書桌工作姿勢頭頂淨空**
    - 書桌面離地 72 cm + 工作坐姿頭頂 ~125 cm → **夾層下緣建議 ≥ 140–150 cm**
    - 需保證下層**能正常使用書桌作為工作 / 閱讀區域**（不會撞頭 + 有充足採光）
  - [ ] **不要壓迫** — 夾層高度讓下方床 + 書桌有足夠淨空
  - [ ] **要明亮** — 下方採光、淺色飾面、燈光（夾層下方需補光，避免書桌區太暗）
  - [ ] **不露雜物** — 儲藏面需有門片 / 布簾 / 推拉門遮擋
- **下層 = 床 + 書桌**（KL 120 Board 翻轉 Murphy Bed）
