## 風格方向

**夾層小空間** — 上層為睡眠區（loft bed），下層整合書桌 + 格狀書櫃 + 收納衣櫃，木色為主 + 白色框架 + 水泥質感。

### 喜歡的細節

- **夾層睡眠區**：抬高後底下空間完整釋放給書桌 + 書櫃
- **格狀方格書櫃**：兼展示與收納，視覺輕量
- **深色踏階 + 白色夾層框**：對比明確，踏階也可藏鞋櫃 / 抽屜
- **自然採光**：側邊長窗引入光線，夾層不會太暗
- **木色 + 白 + 灰**：素雅耐看的三色調
- **金色吊燈**：點綴色溫、避免全冷
