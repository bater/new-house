## 設計決策

### 瓦斯管線 + 熱水器

- [ ] **FN 作為陽台瓦斯管線預留位**，供應：
  - **瓦斯熱水器**
  - **瓦斯洗脫烘洗衣機**（偏好惠而浦滾筒式，見 [F 房三層配置](../rooms/F.md#三層垂直堆疊配置)）
- [ ] **需聯繫瓦斯公司**裝設瓦斯管線（大台北瓦斯 / 欣湖瓦斯等，依本案供氣區域為準）
- [ ] 瓦斯錶位置（表前 / 表後管線走法）與建商 / 管委會協調
- [ ] **管線與電線區隔**（瓦斯管、水管、電源線不可共管槽、需符合法規間距）

### 熱水器

- [ ] **品牌暫定**：**櫻花牌** 或 **Panasonic** — 待定
- [ ] 型號：容量 / 能效等級 / 強制排氣 vs 自然排氣（陽台安裝通常需 FF 強制排氣以確保安全）
- [ ] 安裝高度與排氣方向（需遵循瓦斯熱水器安全規範）
- [ ] 與三層配置（冷氣主機 / 洗衣機 / 貓砂）的空間衝突檢查
- [ ] 冷熱水管路接到室內（廚房水槽 / 衛浴 / 洗衣機）路徑
- [ ] 一氧化碳警報器安裝（法規要求）

### 其他

- [ ] 是否整合 [FE 隱形鐵窗](FE.md) 結構（如果鐵窗延伸到 FN）
- [ ] 外牆開孔協調（瓦斯管、排氣、冷水進氣、熱水出水）— 需與建商 / 管委會確認可施作範圍
