### 投影布幕（AW 側天花預留）

- [ ] **AW 側天花預留投影布幕內嵌盒** — 已列入 [估價 六-5 投影布幕內嵌盒](../contract/estimate/#六木作工程)（9.5 尺 × 850 = NT$8,075）
- [ ] 投影時布幕從 AW 側天花落下，**緊貼 AW 僅留小走道**
- [ ] **使用情境**：投影時廚房停用可接受，但 **水槽 + 冰箱區仍需能取用**
  - 布幕落下位置需稍偏 AW 中央，避開水槽/冰箱正前方
  - 或布幕可分段下降（部分高度）讓水槽區不受影響
- [ ] 投影機位置與布幕距離（需先選機型算投射距離）
- [ ] 電源出口 / 訊號線預埋位置
- [ ] 與 [中央雙吊點](#中央雙吊點多用途懸吊) 不互相干擾

### 中央雙吊點（多用途懸吊）

客廳區天花中央規劃 **兩個結構吊點**，支援多種動態/靜態吊掛用途：

- 空中瑜伽（aerial silk）
- TRX 懸吊訓練
- 懸吊沙發 / 吊床
- 盪鞦韆
- 貓咪爬繩玩具
- 曬衣 / 棉被床單

**結構要求**：
- [ ] **錨點必須鎖入 RC 樓板**，不能只固定於石膏板 / 輕鋼架
- [ ] 建議載重：**靜載 ≥ 300 kg / 錨點**（含動態安全係數，適合空中瑜伽、盪鞦韆）
- [ ] 兩點間距：**80–120 cm**（配合多數 aerial silk / TRX 雙掛點規格）
- [ ] 錨點型式：旋轉 eye bolt 或埋入式鍛造環
- [ ] 不用時可用圓蓋 / 隱藏蓋板遮蔽

**定位**：
- [ ] 位於客廳中央（避開家具動線與主燈位置）
- [ ] 吊掛下方淨空需 **≥ 250 cm**（盪鞦韆動作所需）
- [ ] 需先與 AS 懸吊訓練牆、[AE 冷氣室內機](../walls/AE) 位置協調，避免干涉
