## 房間 / 牆面命名

以房間代號 + 方位命名，共 4 房 × 4 牆 = 16 面牆。

| 房間 | 代號 | 用途 |
|---|---|---|
| A | 客廳 + 廚房 | 公共區 (含前陽台側) |
| B | 主臥房 | 東側 |
| C | 次臥房 | 西側 |
| D | 衛浴 | 中央 |

方位：**N** 北 / **E** 東 / **S** 南 / **W** 西 → 例：`AN` = A 房北牆、`BW` = 主臥西牆。

## 平面概略

北 ↑

```
              ┌──────────── AN ──────────────┐
              │                              │┌────────┐
              │   [廚房]         [客廳]      ││        │
              │                              ││ [前陽台]│
              │     Room A (客廳 + 廚房)     ││ (戶外) │
              │                              ││        │
         AW   │                              │ AE     │
              │              AS              │└────────┘
┌──── CN ─────┼─────────┬──────── BN ────────┤
│             │         │                    │
│             │ 走廊    │                    │
│     C       ├── DN ───┤                    │
│  (次臥)     │         │        B           │
│             │   D     │      (主臥)        │
│             │ (衛浴)  │                    │
│    CW       │         │                    │   BE
│             │         │                    │
└──── CS ─────┴── DS ───┴──────── BS ────────┘

                            南 ↓
```

### 空間關係

- **A 房**寬度 ≈ **D + B**（A 只佔北側的 D + B 區域上方，不涵蓋 C 上方）
- **C 房向西突出**：全棟總寬 **C + D + B > A**
- **C 寬度 > D 寬度**
- **D 南北深度 < C 和 B**：D 較短，北側有空隙
- **走廊** 位於 D 北側的空隙（AS/BN 線以南、DN 以北）
  - C 可經走廊直接通到 B，不必穿過衛浴
- **南牆共線**：CS、DS、BS 同一水平線
