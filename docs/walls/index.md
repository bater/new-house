# 每面牆

20 面牆（5 房 × 4 方位，代號跳過 E）。檔名 = 房間代號 + 方位，例 `AN.md`。

## 房間代號

| 代號 | 房間 |
|---|---|
| A | 客廳 + 廚房 |
| B | 主臥房 |
| C | 次臥房 |
| D | 衛浴 |
| F | 陽台 |

## 方位

**N** 北 / **E** 東 / **S** 南 / **W** 西
