## 設計決策

### 留白留空（通風採光走道）

**CE 不做大型收納或家具**，整面**留白留空**，讓 C 房與 D 之間的走道保持**通風 + 採光**流通。

- [ ] 牆面淺色漆（與 C 房整體木色 + 白框 + 灰水泥調性一致）
- [ ] 不釘牆、不掛厚重裝飾，保持視覺輕量
- [ ] C 房重點收納全集中在 [CW 牆](CW.md)（翻轉床 + 架高夾層上層儲藏）
- [ ] **可兼投影幕面** — 把 A 客廳投影機拿到 C 房時，CE 白牆可直接當幕面（背靠 CW 床躺著看）
