## 天花板：盡量不做

**保留原始樓高最大化**，B 房**盡量不做裝飾天花**：
- [ ] 候選：**直接油漆原始 RC 樓板**（裸頂刷白 / 淺色）
- [ ] 候選：**貼頂照明**（吸頂燈 / 貼頂線燈）— 不壓樓高
- [ ] 冷媒管 / 電線需預留**明管** or **貼頂槽體** 處理方式
- [ ] 與 [F 一對三冷氣](F/#空調系統一對三室外機在-f) 室內機進 B 的管線走法協調（管線藏明管槽 / 牆側收邊）

## 建商窗戶加大（採光通風）

- [ ] 建商原窗**可開啟面積不足**（只有一小段能開）— 採光 + 通風受限
- [ ] **評估更換為更大可開窗面積款式**（例：橫拉改推射 / 增加開啟扇數）
- [ ] **前提**：**不破壞建物外觀**（外框顏色、玻璃樣式、開啟方向盡量維持一致）
- [ ] 候選廠商：原窗同廠商升級，或另找外框可匹配的替代廠商
