## 設計決策

**已確認**：
- ✅ **安裝 [LD002 壁掛翻轉床（雙人）](../references/products/#ld002--壁掛翻轉床雙人--移動式桌--書櫃)** — Hefeng · NT$407,000
  - 床墊 190 × 150 × 18 cm
  - 整體 W184.4 × H220 × D126.6 cm
  - 展開後深度 242.8 cm（含 bed base + 床墊凸出），需確認房間縱深有足夠淨空
  - 床板承重 200 kg；單層書櫃承重 10 kg
  - 兩側整合書櫃（隨主體一體）
- [BW 牆](BW.md) 為衣櫃（Elfa/Boaxel 風格），與 LD002 分工互補

**待確認**：
- [ ] 邊桌配色（白 / 灰）
- [ ] 選配：LED 9k + 床頭片 14.5k + USB 6.5k + 床墊 32.7k
- [ ] BE 壁體結構（RC / 輕隔間需補強）
- [ ] 展開後深度與對面牆淨空（走動動線）

**⚠️ 壁床施工前牆面淨空**（需與水電 + 設計師協調）：
- [ ] BE 牆面**完全清空、貼齊**，不能有凸起 / 掛件 / 飾板影響床身貼牆
- [ ] **原有插座 / 開關面板 封死或移開**，牆面改平
- [ ] **水電重新配線**，將此牆電源移到其他牆（床頭需要的電源保留但避開床身覆蓋區）
- [ ] 床後方若選配 LED / USB，電線預埋到床框固定點後方
- [ ] 水電移位**列入工程追加估價**（目前 [估價單](../contract/estimate/) 未含此項）
