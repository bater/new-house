## 設計決策

**已確認**：
- ✅ **已購 Hefeng KL Board 壁掛翻轉床**（Murphy Bed，詳 [產品頁](../references/products/#kl-board--壁掛翻轉床--書桌murphy-bed)）
  - 床收起時變書桌，翻下變床
  - 需預留 **深度 106 cm** 給書桌翻轉空間
  - 床架離地 26 cm、桌面離地 72 cm

**隔間設計方向**（CW 的上方為儲藏室）：
- [v] **不是臥室** — 純 **收納儲藏** 用途
- [ ] **不要壓迫** — 隔間別做到頂或用沉重實牆
- [ ] **要明亮** — 採光 / 淺色飾面 / 燈光
- [ ] **不露雜物** — 儲物面要有門片 / 布簾 / 推拉門遮擋（開放層板會看到雜物，避免）

**待決**：
- [ ] 選 KL 90 (197×80) 還是 KL 120 (197×104)
- [ ] 是否加購選配：LED 燈、床頭片、原廠床墊
- [ ] 床身結構固定方式（RC / 輕隔間補強）
- [ ] 書桌椅（Murphy 展開後椅子的收納位置）
