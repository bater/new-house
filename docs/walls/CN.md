## 設計決策

### 樓梯櫃（通 CW 架高夾層 + 整體收納）

**CN 規劃為樓梯櫃**：外觀是通往 [CW 架高夾層](../walls/CW.md#架高夾層配置上下層) 上層儲藏的階梯，內部 / 側面整合**抽屜與櫃體**作收納。

- [ ] **雙重用途**：
  - 表面：踏階 → 通往 CW 夾層上層儲藏
  - 內部：每一踏階抽屜 / 踏階下櫃體（收納小型物品、換季衣物、不常用鞋等）
- [ ] 踏階高度 / 深度符合人體工學（踏深 ≥ 25 cm、踏高 ≤ 20 cm）
- [ ] 扶手 / 防護（上層儲藏高度取物安全）
- [ ] 踏階表面材質（止滑）
- [ ] 與 [CW 翻轉床](CW.md) 位置不衝突：樓梯應在 CW 一側（不擋床身翻轉）
- [ ] 與 [CE 留白](CE.md) 的對比：CN 重收納、CE 留白，兩側反差
